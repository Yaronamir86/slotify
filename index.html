קובץ אינדקס מוכן עם רישום ל־Service Worker, מניפסט, אייקונים וחיבור מוכן לפיירבייס.

הקובץ הזה כבר כולל את סקריפט הרישום של ה־Service Worker, מסכי התחברות לבעל עסק וללקוח, וחיבור מוכן ל־Firebase Auth ו־Firestore. את ה־firebaseConfig אתה רק מדביק.

מה יש בגרסה הזו

PWAmanifest + sw + install
Authowner / customer scaffold
Firestorecollections scaffold
Rulesfirestore + storage

הגדרת Firebase

בתוך הקוד קיים אובייקט firebaseConfig עם placeholders. אחרי שתקים App בפיירבייס, הדבק שם את הערכים שלך.

איסוף מומלץ

כדאי להפעיל לפחות Authentication ו־Firestore Database. אם תרצה העלאת לוגו/תמונות, תפעיל גם Storage.

מבנה ראשוני

users/{uid}, businesses/{businessId}, תתי־אוספים של services, employees, bookings.
